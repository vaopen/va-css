## va-theme-chalk 简介

- 通过组合class方式快速开发样式
- 项目采用`scss`, `gulp`构建, 减少了书写大量基本css样式所需要的时间。


## 项目中引入

- `npm install va-theme-chalk`
-  main.ts文件中引入`import 'va-theme-chalk/lib/index.css'`

## 本地构建 ##

```npm
1. npm install

2. npm run docs:dev 本地开发docs

3. npm run build, npm publish 发布包

4. npm run docs:build 打包线上docs
```
